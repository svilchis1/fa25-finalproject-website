    Marriage Data Sources

    • FiveThirtyEight's Marriage & Divorce Rates CSV Collection
      FiveThirtyEight's Github repository is an extensive data collection of marriage and divorce across various demographics. It should be noted that in the data, each individual group has its own percentage that displays unmarried, not married people.
    • ABC's "Marriage Isn't Dead - Yet" Report
      The Github repository is based on this article, claiming that marriage is declining throughout the country, however there are still many people who are getting married.
    • Primary Source Documentation Through IPUMS USA
      The data itself comes from this website, and is a primary source of it all. If needed, this can be checked for accuracy on the data provided for the visualizations and our conclusions.
